高并发预约系统
前端静态页面（无需构建）
登录 用户端 管理员端 库存日志

快速开始

说明：本系统由后端 Spring Boot 直接托管静态页面。登录后会在浏览器本地保存 token，并在请求头中携带 `authorization`。
建议：先打开 登录 获取 token，再访问用户端/管理员端页面。
如果后端部署在不同域名，可能需要后端配置 CORS；但在同域（由同一个后端托管）通常无需额外配置。

接口概览

验证码：`POST /auth/code`
登录：`POST /auth/login`
用户资源：`GET /user/resource/page`
预约：`POST /booking?stock_id=...`
管理员资源：`/admin/resource`（增删改查）
